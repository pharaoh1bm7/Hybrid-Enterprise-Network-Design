## 🚀 Overview
This project demonstrates a hybrid enterprise network that integrates wired and wireless devices with centralized services.

The design focuses on connectivity, basic security, and real-world network structure simulation.

---

## ⚙️ Features

- Centralized Server Infrastructure
- DHCP for automatic IP assignment
- NAT Configuration using ASA Firewall
- Wired & Wireless Network Integration
- Multi-device communication (PCs, Laptops, Smartphones, Printer)

---

## 🧱 Network Design

- Router-based connectivity between network segments
- Switch-based access layer for end devices
- Wireless Access Point for mobile devices
- Central server cluster for services

---

## 🔐 Security

- Basic network protection using ASA Firewall
- NAT for external communication control

---

## 🎯 Objectives

- Build a realistic enterprise-like network
- Practice device connectivity and communication
- Understand integration between wired and wireless environments
- Apply basic network security concepts

---

## 🛠️ Tools Used

- Cisco Packet Tracer

---

## ▶️ How to Use

1. Open the `.pkt` file using Cisco Packet Tracer
2. Test connectivity between devices
3. Check DHCP IP assignment
4. Verify server communication
